# 콘솔 게시판

## 소개
곤솔 게시판

</br>

## 설명
회원가입 후 게시판에 글 올리기 가능. </br>
모든 유저는 게시판의 글쓰기 및 전체 글 읽기 가능. </br>
유저는 자기 자신의 글만 수정/삭제 가능. </br>
Admin 유저는 모든 글을 대상으로 수정/삭제 가능. </br>

</br>

Admin 유저의 정보
- 아이디: admin
- 비밀번호: 1234
